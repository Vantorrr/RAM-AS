# ИНСТРУКЦИЯ ПО ЗАГРУЗКЕ ТОВАРОВ

## Что нужно сделать:

Заполните таблицу Excel по образцу из файла `template_import_products.csv`

---

## ОБЯЗАТЕЛЬНЫЕ КОЛОНКИ (минимум):

1. **name** - название товара (например: "Масло моторное MOBIL 1 5W-30")
2. **part_number** - артикул (например: "550040295")
3. **price_rub** - цена в рублях (например: 3500)

**Остальные колонки можно оставить пустыми.**

---

## ДОПОЛНИТЕЛЬНЫЕ КОЛОНКИ (необязательно):

- **category_name** - название категории ("Масла моторные", "Фильтры воздушные" и т.д.)
- **stock_quantity** - количество на складе (число)
- **manufacturer** - производитель ("MOBIL", "K&N", "NGK")
- **image_url** - ссылка на фото
- **is_in_stock** - в наличии? (TRUE или FALSE)
- **is_installment_available** - рассрочка? (TRUE или FALSE)
- **description** - описание товара

---

## КАК ПРАВИЛЬНО ЗАПОЛНЯТЬ:

### ✅ ЕСЛИ УКАЗЫВАЕТЕ **category_name**:
Просто пишите точное название категории из вашего магазина:
```
Масла моторные
Фильтры воздушные
Тормозные колодки передние
```
→ Товар попадёт точно в эту категорию!

### ✅ ЕСЛИ **НЕ УКАЗЫВАЕТЕ** category_name:
Пишите название товара так, чтобы было понятно что это:
```
✅ "Масло моторное Shell 5W-30" - хорошо
✅ "Фильтр воздушный K&N" - хорошо
❌ "Shell 5W-30" - плохо (не понятно что это масло)
❌ "K&N" - плохо (не понятно что это фильтр)
```

Система сама определит категорию по словам в названии.

## Вопросы:

**- Можно использовать дефис в артикуле?**  
Да, например: "UC3086L-PX8" - это нормально

**- Описание влияет на категорию?**  
Нет, только название товара

**- Какую категорию лучше указывать?**  
Самую точную (подкатегорию), например:
- ✅ "Масла моторные" (точно)
- ⚠️ "Детали для ТО" (слишком общее)
